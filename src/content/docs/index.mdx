## O que é o Helper.DB?

Helper.DB é uma biblioteca JavaScript moderna e poderosa que simplifica o gerenciamento de dados, oferecendo uma API unificada e intuitiva para trabalhar com diferentes tipos de bancos de dados.

## Instalação Rápida

```bash
npm install helper.db
```

## Exemplo Básico

```javascript
const db = require('helper.db');

// Configurar o driver (opcional - padrão é JSON)
db.config({
  driver: 'json',
  path: './database.json'
});

// Operações básicas
db.set('usuario:1', {
  nome: 'João Silva',
  email: 'joao@email.com',
  idade: 28
});

const usuario = db.get('usuario:1');
console.log(usuario); // { nome: 'João Silva', email: 'joao@email.com', idade: 28 }

// Verificar se existe
if (db.has('usuario:1')) {
  console.log('Usuário encontrado!');
}

// Listar todos os dados
const todos = db.all();
console.log(todos);
```
